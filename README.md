# Suggested Data Analytics, OSINT, R, Statistics Readings, and References

A curated reference repository of books, study materials, and technical readings focused on **data analytics**, **open-source intelligence (OSINT)**, **R programming**, **statistics**, **business intelligence**, and related technical disciplines.

This repository serves as a personal knowledge library and structured reading guide for developing stronger analytical, statistical, technical, and investigative skills.

The purpose of this repository is to organize high-value readings and references that support growth in:

- Business Intelligence
